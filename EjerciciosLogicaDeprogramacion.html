Ejercicios Logica de Programacion

Ejercicio 1

programa una función que cuente el numero de caracateres que tiene una cadena de texto, pe. mi_funcion("Hola Mundo") devolverá 10.

Ejercicio 2

programa una función que te devuelva el reverso de una cadena, pe. mi_funcion("Hola Mundo") devolverá

Ejercicio 3

programa una función que dada una cadena de texto y una posición, devuelva el caracter pe. mi_funcion("Hola Mundo")

Ejercicio 4

programa una función que te devuelva una cadena con la primera letra en mayúscula, pe. mi_funcion("hoy es martes") devolverá Hoy es martes

Ejercicio 5

programa una funcion que te devuelva true o false si una cadena tiene o no mayusculas, pe. mi_funcion("Hoy es martes") devolverá true

Ejercicio 6

programa una función que te devuelva el texto recortado según el número de caracteres indicados, pe. mi_funcion("Hola Mundo",4) devolverá Hola

Ejercicio 7

programa una funcion que dada un string te devuelva un array de textos separados por cierto caracter,pe. mi_funcion("hola que tal", " ") devolverá ["hola", "que", "tal"]

Ejercicio 8

programa una funcion que repeta un texto X veces, pe. mi_funcion("Hola", 3) devolverá HolaHolaHola

Ejercicio 9

programa un objeto que permita manejar la siguiente información de una persona: nombre, apellido, edad, dirección, teléfono y fecha de nacimiento.

se debe crear una funcion que permita agregar una nueva persona al objeto.
